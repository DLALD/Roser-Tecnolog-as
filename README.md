# Roser Tecnologías

Sitio web corporativo y marketplace de Roser Tecnologías con productos tecnológicos y servicios de prototipado.

## Características

- Marketplace con modales de productos
- Integración con WhatsApp
- Galería de imágenes interactiva
- Selector de colores y cantidades
- Especificaciones técnicas detalladas

## Productos

- Chasis Gamer Cougar Panzer Evo RGB
- Organizador Magnético de Cables HexaStack H1-80
- Caja Táctica para Munición 9mm
- Organizadores de Cables CCTV (4 y 8 canales)
- Soporte QR para Negocios
- Aplicación 3DCost
- Servicios de Prototipado
